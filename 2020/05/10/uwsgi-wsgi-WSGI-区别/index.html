uwsgi wsgi WSGI 区别

WSGI

英文全称：Web Server Gateway Interface，Web服务网管接口，简单来说它是一种Web服务器和应用程序间的通信规范。

uwsgi

uwsgi是一种通信协议，不过跟WSGI分属两种东西，该协议下速度比较快。

uWSGI

uWSGI是一个Web Server，并且独占uwsgi协议，但是同时支持WSGI协议、HTTP协议等，它的功能是把HTTP协议转化成语言支持的网络协议供python使用。
